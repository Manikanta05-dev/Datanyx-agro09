# VOIS Frontend - React Application

This is the React frontend for the VOIS (Vegan Orbital Intelligence System) application.

## Prerequisites

- Node.js (v16 or higher)
- npm or yarn

## Installation

1. Install dependencies:
```bash
npm install
```

## Development

Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`

## Building for Production

Build the application:
```bash
npm run build
```

Preview the production build:
```bash
npm run preview
```

## Backend Connection

Make sure the Flask backend is running on `http://localhost:5000` before using the frontend.

The frontend is configured to proxy API requests to the backend automatically.
